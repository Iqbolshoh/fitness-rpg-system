# 🏋️ Fitness RPG System

A modern and interactive **Fitness RPG Web Application** built using HTML, CSS, and JavaScript.

This project transforms fitness activities into a role-playing game experience where users earn XP, level up, complete training missions, and track their progress through an engaging and visually appealing interface.

## 🎯 Project Purpose

The goal of this project is to make fitness more engaging by combining workout tracking with RPG-style progression mechanics.

Users can:

* Earn XP through training
* Increase their level
* Complete fitness missions
* Track personal achievements
* Experience a gamified workout journey

## 🎮 Gameplay System

### XP Rewards

| Activity          | XP     |
| ----------------- | ------ |
| Running           | +10 XP |
| Workout           | +20 XP |
| Hardcore Training | +50 XP |

### Level System

* Every 100 XP increases the player's level.
* Progress is displayed visually through the XP bar.
* Players can continuously improve their fitness rank.
